# GhostClaw Autonomous SEO Agents

GhostClaw includes an autonomous marketing system composed of AI agents.

Structure:
SEO CEO → Planner → Research Agents → Content Agents → Builder → Backlink Agent → Analytics

## SEO CEO Agent
Responsible for:
- traffic strategy
- keyword prioritization
- content roadmap

Example decision:
If "AI agent marketplace trending" → create new pages.

## Keyword Research Agent
Discovers:
- keywords
- search volume
- competitor opportunities

## Programmatic Page Generator
Creates thousands of pages automatically.

Templates:
Best AI agents for {industry}
AI agents that {task}
Tools to build {type} AI agents

## Content Writer Agent
Creates long-form articles (2000–4000 words).

Examples:
- What is an AI agent
- How to build autonomous companies

## Marketplace SEO Agent
Optimizes Ghost Mart listings.

Example:
/ghost-mart/agents/seo-agent

## Backlink Agent
Builds authority through:
- GitHub
- developer forums
- AI directories

## Analytics Agent
Tracks:
- traffic
- rankings
- keyword performance
