# GhostClaw Ghost Mart Launch Catalog

This document defines the initial marketplace inventory for Ghost Mart.

Ghost Mart is the capability marketplace of the GhostClaw ecosystem.

Products distributed through Ghost Mart include:

- skills
- agents
- automation packs
- autonomous company blueprints
- developer tools

The goal of this catalog is to seed the ecosystem with installable capabilities that agents and companies can use.

---

# Product Types

Ghost Mart distributes five core product types.

## Skills
Reusable capabilities agents can call.

Examples:
keyword research
SEO analysis
API integration
workflow automation

---

## Agents
Installable autonomous AI workers.

Examples:
SEO CEO Agent
Content Writer Agent
Marketplace Optimizer Agent

---

## Automation Packs
Pre-built multi-agent workflows.

Examples:
SEO Growth Engine
Content Marketing Machine
Lead Generation Automation

---

## Company Blueprints
Entire autonomous businesses that GhostClaw can launch and run.

Examples:
AI SEO Agency
AI Marketing Agency
AI SaaS Factory

---

## Developer Tools
Tools that help developers build agents, skills, and integrations.

Examples:
Agent SDK
Workflow Builder
API Connector Generator
