# GhostClaw System Map

## Purpose

This document provides the **high-level system map** of the GhostClaw platform.

GhostClaw is an autonomous AI operating system designed to:

- run AI agents
- create skills and tools
- operate a capability marketplace
- launch autonomous companies
- expand through a self-growing ecosystem

This file acts as the **top-level blueprint** connecting all GhostClaw architecture documents.

---

# Core Architecture

GhostClaw operates through the following control chain:

Dashboard
↓
API Server
↓
Core Runtime
↓
CEO Engine
↓
Planner
↓
Agents
↓
Skills
↓
Marketplace

This control flow defines how decisions and work propagate through the system.
