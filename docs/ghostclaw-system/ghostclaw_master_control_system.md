# GhostClaw Master Control System

This document explains the entire GhostClaw ecosystem.

## Core Definition
GhostClaw is an Autonomous AI Operating System capable of:
- running AI agents
- creating tools
- launching companies
- operating marketplaces

## Major Components

### Dashboard
UI control panel for monitoring system activity.

### API Server
Handles runtime control, agent registry, and marketplace queries.

### Core Runtime
Runs the heartbeat loop and coordinates agents.

### CEO Engine
Provides strategic planning and task delegation.

### Planner
Converts signals into plans and agent assignments.

### Agents
Autonomous workers performing tasks.

### Skills
Reusable capabilities used by agents.

### Marketplace (Ghost Mart)
Where tools, agents, and skills are distributed.

GhostClaw becomes:
- AI agent operating system
