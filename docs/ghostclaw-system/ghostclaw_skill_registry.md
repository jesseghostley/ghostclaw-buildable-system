# GhostClaw Skill Registry

Skills are reusable capabilities used by agents.

Skills can be installed from Ghost Mart.

---

## Skill Record Structure

skill_id
skill_name
category
description
inputs
outputs
dependencies
install_command
marketplace_status

---

## Example Skill

skill_id: skill_keyword_research
skill_name: Keyword Research
category: SEO

Description:
Discovers keyword opportunities.

Inputs:
seed topic
competitor URLs

Outputs:
keyword clusters
search volume
difficulty scores

Dependencies:
analytics skill
search data API

Marketplace Status:
installable
